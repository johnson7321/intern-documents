1.	下載php
2.	安裝Composer
3.	composer install(安裝 Laravel 專案需要的所有套件，並建立 vendor/ 資料夾和 autoload.php)
4.	修改C:\php-8.4.10-nts-Win32-vs17-x64\php.ini :
	extension=fileinfo
	extension=pdo_sqlite
	extension=sqlite3
5.	建立.env : Copy-Item .env.example .env
6.	設定金鑰: php artisan key:generate
7.	修改.env: DB_CONNECTION=sqlite, DB_DATABASE=C:/project/eggform3/database/database.sqlite
8.	建立sqlite : New-Item -Path . -Name "database.sqlite" -ItemType "File"
9.	安裝node.js
	npm run dev
	npm run build
10.	php artisan serve 連線localhost:8000
